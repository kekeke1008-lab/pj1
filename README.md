# 거래처 재무·뉴스 모니터링

담당 거래처(예: CJ그룹, 대상, 금호타이어, 풍산, 동국계열, 한국도레이첨단소재, 한국바스프, 한국항공우주산업)의
재무정보(DART 전자공시)와 실시간 뉴스(네이버 뉴스검색)를 한 화면에서 모니터링하는 로컬 웹앱입니다.

## 1. API 키 발급 (최초 1회)

### DART 오픈API 키
1. https://opendart.fss.or.kr 접속 후 회원가입
2. 상단 메뉴 "오픈API 이용신청" → 이용자 등록
3. 인증키가 즉시 발급됩니다 (무료, 1일 조회 한도 있음)

### 네이버 검색 API 키
1. https://developers.naver.com/apps 접속 후 네이버 아이디로 로그인
2. "애플리케이션 등록" 클릭
3. 사용 API에서 "검색" 선택, 비로그인 오픈API 서비스 환경 등록 (웹 서비스 URL: `http://localhost` 이면 충분)
4. 등록 후 발급된 Client ID / Client Secret 확인

### 키 입력
`backend/.env.example`을 복사해 `backend/.env`로 만들고 값을 채워주세요.

```
PORT=4000
DART_API_KEY=발급받은_DART_키
NAVER_CLIENT_ID=발급받은_클라이언트_ID
NAVER_CLIENT_SECRET=발급받은_클라이언트_시크릿
```

## 2. 설치 및 실행

```bash
npm run install:all   # 루트/백엔드/프론트엔드 의존성 설치
npm run dev            # 백엔드(4000)+프론트엔드(5173) 동시 실행
```

브라우저에서 http://localhost:5173 접속.

## 3. 사용 방법

1. **설정** 탭에서 담당 거래처를 검색해 추가합니다. CJ그룹, 동국계열처럼 여러 법인으로 구성된
   경우 계열사별로(CJ제일제당, CJ대한통운 등) 각각 검색해서 추가하고, "그룹 라벨"에 같은
   이름(예: `CJ그룹`)을 입력하면 대시보드에서 하나의 그룹으로 묶여 표시됩니다.
2. **대시보드**에서 그룹별 재무 스냅샷 카드와 통합 뉴스 피드를 확인합니다. 뉴스는 10분마다
   자동 새로고침되며, "새로고침" 버튼으로 즉시 갱신할 수 있습니다.
3. 카드를 클릭하면 개별 기업의 재무 추이 차트, 재무제표 상세, 관련 뉴스를 볼 수 있습니다.

## 참고
- 재무 데이터는 DART에 공시된 사업보고서(연간) 기준이며, 연결재무제표가 없는 법인은 개별재무제표로 대체 표시됩니다.
- 비상장 법인(도레이첨단소재, 바스프코리아 등)도 외부감사대상이면 DART에 재무제표가 공시되어 있어 조회 가능합니다.
- DART/네이버 API는 일일 호출 한도가 있으므로, 뉴스는 5분 캐시, 법인코드 목록은 7일 캐시로 호출량을 절약합니다.

## 4. 휴대폰에서 접속하기 (클라우드 배포)

PC를 켜두지 않아도 휴대폰에서 접속하려면 Render(무료)에 배포합니다.

1. 이 저장소를 GitHub에 푸시 (이미 되어 있다면 생략)
2. https://render.com 가입 → "New +" → "Blueprint" → 이 GitHub 저장소 선택
   (저장소 루트의 `render.yaml`을 자동으로 인식합니다)
3. 배포 설정 화면에서 아래 환경변수 값을 입력:
   - `DART_API_KEY`, `NAVER_CLIENT_ID`, `NAVER_CLIENT_SECRET`: 발급받은 키
   - `APP_PASSWORD`: 접속 시 사용할 비밀번호 (외부에 공개되는 앱이므로 필수 설정 권장)
4. 배포 완료 후 발급되는 `https://xxxx.onrender.com` 주소로 휴대폰 브라우저에서 접속
5. 비밀번호 입력 후 로그인 → 브라우저 메뉴에서 "홈 화면에 추가"를 누르면 앱처럼 아이콘이 생깁니다

무료 플랜은 일정 시간 미사용 시 서버가 잠들었다가, 다음 접속 시 첫 로딩에 30초~1분 정도 걸릴 수 있습니다.
